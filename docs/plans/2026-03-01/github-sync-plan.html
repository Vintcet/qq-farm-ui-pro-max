敏感信息检查清单

每次同步前请确认：
  • 检查 .env 文件是否被意外复制
  • 检查 data/*.json 是否包含真实用户数据
  • 检查 data/*.db 数据库文件
  • 检查日志文件是否被包含
  • 检查代码中是否包含硬编码的 API 密钥
  • 运行 grep 搜索敏感关键词
# 敏感信息检查脚本 cd github-sync echo "🔍 检查 API 密钥..." grep -r "sk-[a-zA-Z0-9]\{32\}" . echo "🔍 检查微信 API 密钥..." grep -r "ZoBsM8RqTA326a02tWgQOmOb" . echo "🔍 检查密码哈希..." grep -r "adminPasswordHash.*[a-f0-9]\{64\}" . echo "🔍 检查数据库文件..." find . -name "*.db" echo "✅ 检查完成"
