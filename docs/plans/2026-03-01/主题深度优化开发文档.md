## 项目概述

### 背景

当前 QQ 农场助手 UI 系统已具备 5 套主题（御农翠绿、樱花粉黛、全息赛博、尊贵黯金、深海矩阵），但主题切换仅影响：
- ✅ 左侧边栏部分元素
- ✅ 少量文字颜色
- ✅ 基础主色调

**问题：** 其他 4 个主题缺乏深度联动，整体界面氛围感不足，无法给用户带来沉浸式的视觉体验。

### 优化范围

| 主题名称 | 主题 Key | 主色调 | 当前完成度 | 目标完成度 |
|---------|---------|--------|-----------|-----------|
| 御农翠绿（默认） | `default` | #22c55e | ✅ 100% | ✅ 保持现状 |
| 樱花粉黛 | `sakura` | #ffc0cb | ⚠️ 30% | 🎯 100% |
| 全息赛博 | `cyber` | #8b5cf6 | ⚠️ 30% | 🎯 100% |
| 尊贵黯金 | `elegant` | #eab308 | ⚠️ 30% | 🎯 100% |
| 深海矩阵 | `ocean` | #0ea5e9 | ⚠️ 30% | 🎯 100% |

## 技术方案

### 技术栈

- **框架：** Vue 3.4+
- **状态管理：** Pinia 5.x
- **样式方案：** CSS Variables + UnoCSS + 原生 CSS
- **动画引擎：** CSS Keyframes + CSS Transitions
- **工具库：** @vueuse/core

### 核心架构

```
┌──────────────────────────────────────────────────────────┐
│                     主题系统架构                          │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌─────────────┐     ┌──────────────┐     ┌──────────┐ │
│  │  Pinia Store │ ──▶ │ CSS Variables │ ──▶ │  组件样式  │ │
│  │  colorTheme  │     │  :root / .theme │     │  动态绑定  │ │
│  └─────────────┘     └──────────────┘     └──────────┘ │
│         │                   │                    │       │
│         │                   │                    │       │
│         ▼                   ▼                    ▼       │
│  ┌─────────────┐     ┌──────────────┐     ┌──────────┐ │
│  │ 主题状态管理  │     │ 主题调色盘    │     │ 动态背景  │ │
│  │ localStorage│     │ 渐变/氛围色   │     │ Mesh/动画 │ │
│  └─────────────┘     └──────────────┘     └──────────┘ │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### CSS 变量层级结构

```css
/* Level 1: 基础色阶（已存在） */
--color-primary-50 ~ --color-primary-950

/* Level 2: 背景系统（新增） */
--theme-gradient-start   /* 渐变起始色 */
--theme-gradient-mid     /* 渐变中间色 */
--theme-gradient-end     /* 渐变结束色 */
--theme-bg               /* 纯色背景备用 */
--theme-dark-bg          /* 深色模式背景 */

/* Level 3: 氛围系统（新增） */
--theme-glow-color       /* 发光色 */
--theme-soft-shadow      /* 柔和阴影 */
--theme-card-bg          /* 卡片背景（半透明） */
--theme-card-border      /* 卡片边框 */

/* Level 4: 强调色（新增） */
--theme-accent-1         /* 强调色 1 */
--theme-accent-2         /* 强调色 2 */
```

## 性能优化指南

### 动画性能

**✅ 推荐做法：**
```css
/* 使用 transform 和 opacity（GPU 加速） */
@keyframes float {
  0%, 100% {
    transform: translate(0, 0) scale(1);
    opacity: 0.8;
  }
  50% {
    transform: translate(10px, -20px) scale(1.1);
    opacity: 1;
  }
}

/* 使用 will-change 提示浏览器 */
.glow-orb {
  will-change: transform, opacity;
}
```

**❌ 避免做法：**
```css
/* 避免动画这些属性（触发重排） */
@keyframes bad-animation {
  0% {
    width: 100px;      /* ❌ 触发重排 */
```

### C. 常见问题 FAQ

**Q1: 主题切换时页面闪烁怎么办？**

A: 确保背景层和内容层都设置了 `transition: background 0.5s ease`，并使用 `will-change` 提示浏览器。

**Q2: 毛玻璃效果在部分浏览器不显示？**

A: 添加 `-webkit-backdrop-filter` 前缀，并为不支持的浏览器提供降级方案（纯色背景）。

**Q3: 移动端性能问题如何解决？**

A: 实施移动端降级策略：关闭 Mesh 背景、降低模糊半径、减少粒子数量、延长动画时间。

**Q4: 如何测试动画 FPS？**

A: 打开 Chrome DevTools → Performance → 录制 → 查看 FPS 图表。
