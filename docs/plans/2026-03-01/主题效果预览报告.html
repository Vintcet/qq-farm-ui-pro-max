背景动画演示

每个主题拥有独特的背景动画方案，包括 Animated Gradient 和 Mesh Gradient 两种技术实现。

🌸 樱花粉黛 - Mesh Gradient

3 个粉色光球缓慢浮动（20s 循环）

特效： 樱花飘落粒子（可选）

⚡ 全息赛博 - Animated Gradient

快速流动的紫蓝渐变（10s 循环）

特效： 网格线条 overlay + 霓虹闪烁

👑 尊贵黯金 - Mesh Gradient

极慢速的金色光球（30s 循环）

特效： 金粉闪烁 + 边框流动

🌊 深海矩阵 - Animated Gradient

波浪状青蓝渐变（12s 循环）

特效： 气泡浮动 + 水波纹点击

开发进度计划

整个开发周期预计 8-12 小时，分为 4 个阶段实施。

2-3 小时

阶段一：CSS 变量系统升级

为 4 个主题定义完整的 CSS 变量调色盘，包括渐变背景、氛围色、强调色等。

  • 扩展 theme-sakura 变量组
  • 扩展 theme-cyber 变量组
  • 扩展 theme-elegant 变量组
  • 扩展 theme-ocean 变量组
  • 添加全局动画 Keyframes
  • 添加毛玻璃工具类
3-4 小时

阶段二：动态背景系统

实现 Animated Gradient 和 Mesh Gradient 两种背景方案，并集成到 App.vue。

  • 创建 ThemeBackground 组件
  • 实现 Animated Gradient（Cyber/Ocean）
  • 实现 Mesh Gradient（Sakura/Elegant）
  • 集成到 App.vue
  • 移动端性能降级策略
4-5 小时

阶段三：组件联动改造

对核心组件进行深度主题化改造，实现全组件联动效果。

  • Sidebar.vue 深度优化
  • Dashboard.vue 卡片改造
  • BaseButton 渐变 + 光晕
  • BaseInput 聚焦特效
  • LandCard 主题边框
  • FarmPanel 毛玻璃背景
2-3 小时

阶段四：特效增强

为每个主题添加专属特效，并进行性能优化。

  • Sakura：樱花飘落粒子
  • Cyber：网格线条 + 霓虹闪烁
  • Elegant：金粉闪烁
  • Ocean：气泡浮动
  • 移动端适配优化

验收标准

🎯 功能测试

  • 主题切换平滑无闪烁
  • 背景动画流畅（FPS > 50）
  • 深色模式正常显示

  • localStorage 状态保持

  • 渐变平滑无色块分界
  • 光晕效果柔和不刺眼
