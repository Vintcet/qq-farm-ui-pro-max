四、 风险预案 (Impact Analysis)

1. 命令失效：由于移动了脚本文件，如使用到了 ai-autostart.js 的某些古老自动化挂载脚本可能找不到路径。我会在移动前检查 start-with-ai.sh 的实际调用情况。
2. Git 记录膨胀：大量的文件移动（Rename 操作）会产生 Git 树变动，不过由于系统当前未能关联上 Git（在你的环境下 fatal: not a git repository），可以直接进行物理层剪切覆盖移动。
