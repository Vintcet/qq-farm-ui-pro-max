🚀 近期优化建议采纳及影响消除计划

本文档响应《API密钥集中管理与近期优化核查报告》中的安全、性能与交互建议，实施排期规划。

模块一：API 密钥网络脱敏传输防护 Priority: High (P1)

现状痛点： `Settings.vue` 以密码框 `type="password"` 渲染第三方 API (如 `wxApiKey`)。但在初始化 `loadThirdPartyApiConfig()` 拉取配置经过网络层时，密钥作为 JSON 明文响应，存在被中间件探嗅的越权风险。

修复策略与机制 (Masking Strategy)：

  • 后端切流 (Masking)： 在 `admin.js` 中 `GET /api/admin/third-party-api` 输出前，运用正则对密钥仅保底渲染：`sk-**** + 末尾4位字符`。
  • 前端回传修正 (Diff Upload)： 当超级管理员点击“保存”时，前端在 `saveThirdPartyApiConfig` 请求中检测如果字符串依旧包含 `****`，则将其从 HTTP Body (Payload) 剔除，向后台发出指令**“保持原本值，无需覆盖”**。
  • 后端安全拼接 (Patch Merge)： 后台 `POST /api/admin/third-party-api` 接口接收到 Payload 时，遇到未被传参的 `wxApiKey`，直接合并保留原先 `store.json` 里未加密的持久化值。

操作边界：此修正完美达成安全审计。用户不修改时密码不流出；只有用户清空重输了一个完整的 Key（即没有星号时），它才会被真实传输并落库覆盖旧值。

模块二：登陆界面（核显杀手）极限性能隔离 Priority: High (P1)

现状痛点： 经过 v3.9.0 极品流明材质与 `cubic-bezier` 弹簧特效重组的 `Login.vue`，在搭载上代核显设备的 Chrome 渲染环境中，由 `hover:scale-105` 及 `animate-pulse` 等 GPU 加速合成引发重绘负荷（风扇升温）。

修复策略与机制 (Performance Override)：

  • 扩大覆盖面： 将位于全局 `style.css` 或 `App.vue` 内原本专门为内部组件兜底的性能开关 `.performance-mode-active` 选择器边界扩大，包容最高入口的 `Login` 父挂载点。
  • 原子类硬拦截： 在极简性能开关接管时，利用 CSS 层级直接注入 * { transform: none !important; animation: none !important; transition: none !important; }，一剑斩断登录底板星空粒子和弹簧缩放 3D 转换（`translate3d / scale`），大幅降维。

模块三：智能抗边缘遮挡体系 (Floating UI) Priority: Medium (P2)

现状痛点： 虽然 v3.9.0 已统一将所有的 `Tooltip` 从侧边防割裂改为下方居中弹出。但在纵向空间极端紧缺（此时处于屏幕最后 30px）的环境下，往下弹会无视高度突破 `body` 底边，致使整个页面出现极长的滚动条或被遮住。

修复策略与机制 (Dynamic Boundary Calculation)：

  • 放弃纯 CSS 的被动站位： 修改 `BaseTooltip.vue` 中单纯的 top-full 定位类。
  • 引入动态交会 (Intersection Strategy)： 在挂载后或者 Hover 前触发轻量级 JS `$el.getBoundingClientRect()` 测算。
  • **智能倒转：** 判定 `Y 轴剩余视口 (window.innerHeight - rect.bottom) < 气泡预估高度 (约 40px)` 时，自动切换为向顶部弹射 `bottom-full mb-1` 并微调阴影方向，彻底消灭界面撑界 Bug。

执行计划总览 & 影响验收 (Roadmap)

  • Phase 1: 攻克【模块一】，改写后端 `/api/admin/third-party-api` 的 GET 遮蔽及 POST 合并，保障配置不落库失效。 (预计投入 15 分钟)
  • Phase 2: 收缩核显压力【模块二】，通过扩充 `.performance-mode-active` 到 Root 或在 Login.vue 中引入状态，实现物理切割。 (预计投入 5 分钟)
  • Phase 3: 导入核心交互演算【模块三】，升级 `BaseTooltip.vue` 追加空间边界反馈计算函数。 (预计投入 15 分钟)
