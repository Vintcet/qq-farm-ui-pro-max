# 排行榜配色与主题联动优化计划

## 背景与目标
当前排行榜（LeaderboardModal）的配色与系统主题色融合度不足，视觉层次不够明晰。本次优化旨在通过非侵入式开发，使其配色与全局变量 `--color-primary-xxx` 深度绑定，并提升玻璃拟态交互质感。

## 关键设计变更
1. **深度变量联动**：
   - 移除部分硬编码的 Tailwind `primary` 类（若其未映射至变量），确保所有核心元素通过 `rgb(var(--color-primary-xxx))` 系列变量渲染。
   - 调整排头（Header）与列表头（Thead）的透明度与背景色，使其更具“通透感”。

2. **排名视觉重塑**：
   - 前三名的奖牌样式调整为带有微弱内发光的细腻材质感，而非单纯的色块。
   - 非前三名的数字采用更柔和的变量色。

3. **数据列配色优化**：
   - 金币与点券颜色调整，使其在暗色模式和亮色模式下均保持优雅且不突兀。
   - 增加行悬浮（Hover）时的主题色微亮反馈。

4. **状态指示器同步**：
   - “在线”状态圆点颜色与主题核心色保持一致（或配合绿色的主题变体）。

## 验证计划
- **主题切换测试**：在不同主题（如赛博、深海、樱花）间切换，检查排行榜主色调是否随之精准变化。
- **模式对比测试**：验证在暗黑模式（Dark Mode）与亮色模式下的可读性。
- **交互自测**：检查滚动条样式与悬停动效是否与系统全局保持一致。
