# 排行榜配色优化报告

## 优化概述
针对排行榜组件（LeaderboardModal）进行了深度的视觉重塑，核心在于将原本孤立的配色方案与系统全局主题变量（`--color-primary-xxx`）进行深度解耦并重连。

## 详细优化项
1. **主题色彩深度联动**：
   - 将所有带有业务逻辑色彩的元素（如状态指示器、悬停反馈、排头背景等）全部替换为基于 `primary-500` 的透明度变体（如 `bg-primary-500/10`）。
   - 实现了“状态同步”：账号在线状态的圆点现在会随着系统主题色的变换而同步改变。

2. **排名质感提升**：
   - 为前三名（1st, 2nd, 3rd）设计了具有金属质感的线性渐变奖牌样式，辅以 `shadow-lg` 立体投影和半透明 `border` 描边。
   - 移除了原本扁平的色块，增加了视觉上的尊贵感。

3. **玻璃拟态层次加强**：
   - 调整了 Modal Header 与 Table Header 的 `backdrop-blur` 级别与背景透明度，使其在各类复杂背景下（如自定义壁纸或网格背景）具备极佳的可读性。
   - 优化了数据行（Row）的悬停反馈，使用 `hover:border-primary-500/20` 进行精致边框勾勒。

4. **滚动条与微动效**：
   - 滚动条宽度微调至 5px，配色同步为主题半透明色，降低了对界面的视觉干扰。
   - 在线状态增加了 `animate-pulse` 呼吸效果。

## 验证结果
- **非侵入性确认**：所有修改均限制在 `LeaderboardModal.vue` 内部，未触及全局 CSS 或后端逻辑。
- **主题兼容性**：在所有内置主题（御农翠绿、全息赛博、尊贵黯金等）下均表现一致且优雅。

请通过界面右上角的排行榜按钮打开并检查优化后的高级感。
