# 优化登录页

## 目标
优化登录页面的视觉体验，包括名字修改、增加程序特点、悬停动画，以及非侵入式的整体验配色优化。

## 系统设计与边界
基于现有 `web/src/views/Login.vue`，仅使用 Tailwind CSS 与内联的 Scope CSS，不引入新的第三方动效库，不修改核心逻辑。

## Proposed Changes

### Login Component
- 调整页面主体背景与左侧信息栏（品牌展示区）配色，使之更有现代感与深邃感（例如带有微弱的紫蓝色过渡）。
- 更改系统名称：从 `QQ 农场智能助手` 改为 `御农·QQ 农场智能助手`，并为其加入动态的光影扫过或渐变呼吸效果。
- 扩充程序特点标签：除原有的外，新增“无感智能验证码识别”、“多核高并发执行引擎”等标签，增强产品高级感。
- 对新增的特性标签使用 `cubic-bezier(0.34, 1.56, 0.64, 1)` 实现高级动态弹簧阻尼悬停动画。
